01 / CAD desktop

MadCAD 2D

Aplikacja do szybkiego projektowania elementów technicznych i konstrukcji stalowych. Łączy klasyczne narzędzia CAD z gotowym workflow dla bram, ogrodzeń i balkonów.

  • Rysowanie, wymiarowanie, warstwy i edycja obiektów
  • Generatory konstrukcji oraz eksport DXF, SVG i PDF
  • Gotowe wydania dla macOS ARM64 i Windows x64

02 / Mobile i desktop

MadCalc

Lokalny kalkulator optymalizacji cięcia sztang. Wylicza plan rozkroju, zużycie materiału i odpad, a gotowy wynik zapisuje w czytelnym raporcie PDF.

  • Obliczenia działające całkowicie lokalnie, bez backendu
  • Natywna aplikacja iOS oraz wersja wieloplatformowa
  • Eksport PDF i automatyczne zapisywanie stanu
SwiftUI Flutter Dart GitHub Actions
